# StudyRoom - Premium Room Booking Platform

**StudyRoom** is a modern, full-stack web application built for booking and managing study rooms, meeting spaces, or rental rooms. It features a sleek dark UI, real-time room management, and a smooth booking experience.

## ✨ Features

- **Browse & Book Rooms** — Discover beautifully designed study rooms with detailed information, images, and amenities.
- **Smart Room Management** — Admins and owners can easily add, edit, and delete rooms with a powerful dashboard.
- **Real-time Booking System** — Seamless booking experience with availability checking.
- **Responsive & Modern Design** — Built with Next.js 16 and beautiful dark theme using HeroUI components.
- **Amenity-rich Rooms** — Each room includes WiFi, AC, TV, Kitchen, and other premium facilities.
- **Secure Authentication** — Protected routes and user session management.
- **Fast Performance** — Powered by Next.js App Router, Turbopack, and optimized rendering.

## 🛠️ Tech Stack

- **Frontend**: Next.js 16 (App Router), React, Tailwind CSS
- **UI Library**: @heroui/react
- **Icons**: React Icons
- **Notifications**: React Toastify
- **Backend**: Next.js API Routes + External API
- **Database**: MongoDB (assumed from project structure)
- **Deployment**: Vercel / Netlify / Custom Server
